與[ProjectDivaControllerServer](https://github.com/JamilHsu/ProjectDivaControllerServer)配套一同使用的安卓端程式。可以把平板/手機變成*初音未來:歌姬計畫*的控制器，享受比Switch版的「觸控遊玩」還要更好的遊戲體驗。

使用方法: 於電腦上啟動ProjectDivaControllerServer後在第一個欄位中填入電腦的IP位址並點擊連線Connect按鈕即可。  
有一個自動搜尋伺服器(電腦端)IP位址的按鈕，不過並不是在所有情況下都有效，此時就必須自行手動輸入IP。  
更多詳細資訊請參閱[ProjectDivaControllerServer](https://github.com/JamilHsu/ProjectDivaControllerServer)
